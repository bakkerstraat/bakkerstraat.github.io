## Metal-Crypto Tracker

Interactive web application for tracking precious metals and cryptocurrency prices.

### Features
- Real-time price data for gold (XAU/USD), silver (XAG/USD), and Bitcoin (BTC/USD)
- 30-day historical charts with smooth animations
- Individual and combined view modes
- Responsive design

### Data Sources
- **Metals**: [GoldPriceZ API](https://goldpricez.com)
- **Crypto**: [CoinGecko API](https://www.coingecko.com)

## Tech Stack

- HTML5/CSS3
- JavaScript (ES6+)
- Chart.js for data visualization
- GitHub Pages for hosting
